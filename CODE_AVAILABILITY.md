# Code Availability

The analysis code used for data processing, model construction, validation,
visualization, and reproducibility checks is available in this repository.

Input data are not redistributed here. The scripts use public datasets from
UCSC Xena, GEO, cBioPortal/GDC CPTAC, and GDSC2; users should obtain those data
from the original repositories and place them under the expected `data/raw/`
structure before running the workflow.

After publication, a stable archived release can be created through Zenodo and
cited with its DOI.
